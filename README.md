# ちょい捨て（最小構成）

初心者向けに、**今日のタスクを1つ表示するだけ**の最小アプリです。

## 起動方法（ローカル）

1. このフォルダに移動
2. 以下を実行

```bash
python3 -m http.server 8000
```

3. ブラウザで次を開く

```text
http://localhost:8000
```

## ファイルの役割

- `index.html`
  - 画面の骨組み（タイトル、タスク表示、ボタン）を定義します。

- `style.css`
  - 見た目（余白、色、カード、ボタン）を整えます。

- `app.js`
  - タスク一覧を持ち、ランダムに1つ表示します。
  - 「別のタスクにする」ボタンを押すと、別タスクに切り替えます。
  - 「完了」ボタンを押すと「完了しました」と表示し、履歴に追加します。

- `ui_copy_choisute_v1.md`
  - 先に作成したUI文言仕様です。今後機能を増やすときの元資料です。

## main 競合が出たときの更新手順（メモ）

手動で競合を直さず、最新 `main` を取り込んだうえで変更を再適用する場合の例:

```bash
git fetch origin main
git reset --hard origin/main
git cherry-pick <完了機能のコミット>
```
